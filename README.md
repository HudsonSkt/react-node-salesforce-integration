# 🚀 Full Stack Application - React + Node.js + Salesforce Integration

## 📌 Sobre o Projeto

Aplicação Full Stack desenvolvida com React (TypeScript + Vite) no frontend e Node.js no backend, integrando dados da plataforma Salesforce através da biblioteca JSForce.

O projeto foi desenvolvido com foco em arquitetura cliente-servidor, organização de código, componentização e integração com APIs REST externas.

---

## 🛠️ Tecnologias Utilizadas

### 🔹 Frontend
- React.js
- TypeScript
- Vite
- Styled Components
- Axios

### 🔹 Backend
- Node.js
- Express
- JSForce
- API REST

---

## 🔗 Funcionalidades

- Consumo de APIs REST
- Integração com Salesforce
- Comunicação entre frontend e backend
- Componentes reutilizáveis
- Estilização com Styled Components

---

## 🏗️ Arquitetura

frontend/ → Aplicação React (Vite + TypeScript)  
backend/ → API Node.js responsável pela integração com Salesforce  

---
## ▶️ Como Executar o Projeto

### 🔹 Backend

```bash
cd backend
npm install
npm run dev
```

### 🔹 Frontend

```bash
cd frontend
npm install
npm run dev
```
